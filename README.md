# ねぇ、私たち付き合っちゃう？

入力した言葉をきっかけに画像と会話が切り替わる、短編の恋愛シミュレーションゲームです。

[ブラウザでゲームをプレイ](https://yamamotokensuke0301.github.io/nee-tsukiattyau/)

## 現在のゲーム進行

1. タイトル画面で「はじめる」を選ぶ
2. 画像1と「ニシシ はじめまして」を表示する
3. プレイヤーが任意のセリフを入力する
4. 入力のたびに、直前と同じ見た目を除く画像1〜8から一枚を抽選する
5. 画像2〜8では「うん」と返し、入力へ戻る
6. 画像1が再表示された場合は「ねぇ 私たち付き合っちゃう？」と尋ねる
7. 「はい」でエンディング、「いいえ」でランダム表示のループへ戻る

画面切り替え時には短い効果音を鳴らします。タイトル画面は文字だけで構成し、エンディングでは白一色の画面に「製作者：空希 香蕉（そらき ばなな）」だけを表示して、`assets/ending-theme.mp3` を再生します。製作者表記は映画のクレジットのように、曲の再生時間に合わせて上の画面外から静かに現れ、一定速度で画面を縦断して、曲の終了時に下の画面外へ消えます。

## BGM

画像1〜8にはそれぞれ固有のループBGM（`assets/bgm/heroine-01.m4a`〜`heroine-08.m4a`）が付き、画像の切り替えと連動してクロスフェードします。画像1のBGMは最初の「ニシシ はじめまして」でだけ流れ、抽選で画像1へ戻った告白場面では無音になります。エンディングに入るとエンディング曲だけが流れます。

- 画像1: 「はじめましての余白」— ピアノ、ナイロンギター、薄い弦
- 画像2: 「振り返る風」— ナイロンギター、ベース、フルート
- 画像3: 「縁側、午後三時」— エレクトリックピアノ、ビブラフォン
- 画像4: 「バスはまだ来ない」— 点描的なピアノ、クラリネット
- 画像5: 「夕暮れの水音」— 6/8風のギターとピアノ
- 画像6: 「花を見ているふり」— 小さなギターとオルゴール
- 画像7: 「ノートの端の落書き」— エレクトリックピアノとピチカート
- 画像8: 「夏空へ」— 上昇するピアノ、弦、フルート

BGMはApple標準のGeneral MIDI音源をAVFoundationでオフラインレンダリングし、会話を邪魔しない音圧へ曲ごとに正規化します。再生成にはmacOSが必要です。

```sh
swift tools/render_bgm.swift              # 全8曲
swift tools/render_bgm.swift --only 01 06 # 指定曲だけ
```

画像4・6・8は、横顔、高角度の伏し目、低角度の見上げという異なる構図で新規生成した画像へ差し替えています。

## 起動方法

静的ファイルだけで動作します。プロジェクトのディレクトリでローカルサーバーを起動してください。

```sh
python3 -m http.server 5175
```

ブラウザで `http://127.0.0.1:5175/index.html` を開きます。
